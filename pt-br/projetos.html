Sobre os projetos

Antes de mais nada, é importante deixar claro que todos os projetos aqui apresentados são de caráter acadêmico e demonstrativo. Desenvolvidos utilizando HTML5, CSS3, JavaScript, Bootstrap 4 e 5, adaptados para todos os dispositivos e baseado nos conceitos de design UI & UX.

Conceitos

UI significa User Interface, o que remete a interface gráfica apresentada ao usuário, que por meio dela se comunica com o site ou software de maneira intuitiva.

UX significa User Experience, que traz sentimento às experiências do usuário que está utilizando a interface e navegando pela plataforma digital.
