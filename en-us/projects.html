About the projects

First of all, it is important to make it clear that all the projects presented here are of an academic and demonstrative character. Developed using HTML5, CSS3, JavaScript, Bootstrap 4, adapted for all devices and based on UI & UX design concepts.

Concepts

UI stands for User Interface, which refers to the graphical interface presented to the user, who through it communicates with the website or software in an intuitive way.

UX stands for User Experience, which brings feeling to the experiences of the user who is using the interface and navigating into the digital platform.
